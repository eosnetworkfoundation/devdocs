## Description

Returns data about a particular name and it's bidding status

## Positionals
* `name` _TEXT_  - The name to lookup

## Options
* `-h,--help` - Print this help message and exit
* `-j,--json` - Print result as json

## Usage

```sh
cleos system bidnameinfo bob
```
