## Description

Locks a wallet

## Positionals
None

## Options
- `-n, --name` _TEXT_ - The name of the wallet to lock
## Usage


```sh
cleos wallet lock
```
or
```sh
cleos wallet lock -n second-wallet
```

## Outputs

```console
Locked: 'default'
```
